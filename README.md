# HealthCare-Data-Anaalysis

### **Project Description**

This project involves comprehensive analysis of healthcare data using Microsoft Excel to derive meaningful insights into patient demographics, medical conditions, and hospitalization patterns. The dataset consists of multiple tables, including healthcare records, medical examinations, and hospitalization details, which were cleaned, structured, and integrated for analysis.

Key Excel techniques such as data cleaning, lookup functions, pivot tables, and conditional formatting were applied to transform raw data into a structured format. Interactive dashboards and visualizations were created to highlight trends, relationships, and key performance indicators. The analysis focuses on understanding how factors like age, BMI, and city tier influence health conditions and hospital admissions.

---

### **Project Summary**

* Cleaned and consolidated multi-sheet healthcare data for analysis
* Performed exploratory data analysis on patient demographics and medical metrics
* Built pivot tables and dashboards to visualize key healthcare trends
* Identified relationships between BMI, age, and hospitalization patterns
* Delivered actionable insights to support healthcare decision-making
